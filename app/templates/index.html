gRNAdb is a free database that permits the user to evaluate the effectiveness of a gRNA across the human exome. The database was compiled from the implementation of off target and published on-target scoring systems, "link to papers here". The database also contains information regarding the Conservation score, position of gRNA, single nucleotide polymorphism (SNP) count and the presence of a TTTT repeat, which can be important for selecting an effective gRNA

In order to extract relevant gRNA sequences, the user can search the database using the gene name and specify extra parameters where gRNAs can be selected based on parameters such as Constitutively active exons, strand and the presence of TTTT repeats in the gRNA
