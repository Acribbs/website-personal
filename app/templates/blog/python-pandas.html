THE PANDAS MODULE FOR PYTHON

  • Adam Cribbs
  • March 21, 2017

Im still relatively new to python programming (well two years fresh) and one of the things I love about python is the ability to easily import modules that give you extra functional code with which to use and make simpler programs.

Im sure a lot of you reading this will have come across the pandas module, if you have then this post isn’t for you, if you haven’t then please keep on reading as it has completely revolutionised the way I work and I would like to share my reasons for investing time to learn how this module operates.

I initially became interested in pandas because I was getting sick of loading data to a database and then using SQL to perform joins. A quick Google search revealed that this could be performed more easily in python, using the module pandas, prior to loading the data into the database.

Obviously working with databases is easy and doesn’t require any extensive training courses, however I find the code to search databases boring to write and found it cumbersome when integrating it into my python code. However, the discovery of pandas has revolutionised the way I interact with data in python.

Alongside Numpy, Pandas is now, without a doubt, one of the most important Python libraries for data analytics. Easy to use data structures (DataFrame, Series are the most important ones), similar to R in the fact that it uses data frames to manipulate data, markedly improves the functionality of python code.

To help new users find information regarding the best resources on where to start to learn pandas I have compiled the following list of references in order of usefulness (At least to me). However if you are really interested in exploring pandas fully, I recommend you read the docs because there is so much functionality in this module:

  1. The official documentation and tutorials from pandas developers – this is a great resource and introduces the principles of pandas.
  2. The Thomas Wiecki blog – A really easy to follow first introduction example
  3. Coursera on working with data in python - I took this course provided by the University of Michigan and found some of their pandas introductions useful.
  4. Pandas Tutorial: DataFrames in Python - I found this course very useful for understanding small chunks of code because of the interactive learning platform.

All in all, it took me a month to have a good grasp of python and about another month to start using the language to a sufficiently advanced level.
