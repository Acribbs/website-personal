IPYTHON NOTEBOOK

  • Adam Cribbs
  • March 21, 2017

This is the first tutorial I have for my blog and I plan on going through the basics of ipython notebook because I have found the ability to run snippets of code very useful for writing and testing code.

IPython notebook can easily be installed using the following:

pip install "ipython[notebook]"

However, I would recommend using conda for installation because it allows you to easily manage your python environment using a few simple commands. A conda environment can be installed by following the instructions here.

To find whether ipython is currently installed on conda environment

Once you have installed ipython then it is as simple as change directory into the desired folder you want to create notebooks in then run the command:

ipython notebook

You should see the following information during the startup:

A browser should open with the opening iptyhon notebook page. If this does not happen then direct your browser to http://localhost:8888/.

To create a new ipython notebook click new in the top right hand corner and select a python environment and a new window should open with an empty cell.

Each IPython notebook consists of cells that can contain either a heading, Markdown or python code.

When you have written some code you can press shift-Enter to execute the code. This will generate an output cell displayed beneath the input cell. For example running the first code cell will generate an output like this:

Convention dictates that the cells are ran from top to bottom, this can produce unintended consequences if you run cells out of sync because of the inheritance of global variables. Be aware of this issue and try and write code that uses unique variable names and try to avoid inheritance as much as possible throughout the notebook.

After you have finished writing your code save it and you can return to it at a later date.

This is just a quick introduction to using ipython notebook, for more information on setting up an environment for ipython and more on R magic please click here for my other post of more advanced features.
