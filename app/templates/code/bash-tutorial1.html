BASH/SHELL SCRIPTING

  • Adam Cribbs
  • April 7, 2017

It's always been a difficult task trying to find good resources to understand bash scripting. Although bash scripts are very useful and are critical for running parallelised tasks in a high performance cluster (HPC), I have always struggled finding resources that are in-depth enough to understand some of the advanced functionality that you can gain with bash/shell scripting.

Unlike python and R, where there are fantastic online tutorials, the main resource that seems to be useful is youtube. Specifically, two main resources:

1. Derek Banas

The link to the tutorial can be found here. I like this tutorial because its only 1 hour, has accompanying code that can be saved to your desktop and used as a reference, plus is really easy to follow!

2. The Bad Tutorials

The link to the tutorial can be found here. This is a 62 part tutorial and goes into some of the more advanced functionality of shell scripting. I really like this tutorial because it's easy to follow and you can pick and choose the clips you are most interested in as they are labelled well.
